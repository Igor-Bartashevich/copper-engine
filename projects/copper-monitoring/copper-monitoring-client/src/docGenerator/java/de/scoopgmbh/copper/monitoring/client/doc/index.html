Copper Monitor

Copper Monitor is a javafx desktop application that provides access to Copper Workflow Engine. It includes workflow management, workflow instance inspection and performance statistics.

Quickstart

  • svn: http://copper.googlecode.com/svn/trunk/ (read-only)
  • gradlew eclipse
  • eclipse project import
  • example start: MonitoringExampleMain.java
  • client start: MonitorMain.launch
