Dashboard

Introduction

Dashboard shows the configuration of copper. The dashboard view supports adjustment of processorpool threadpool size, threadpool priority and the batcher threadpool size

Example

If the engine have 2 pools the gui displays 2 pooltabs.

For the input: 7 copper will set pool size to: 7

For the input: 3 copper will set pool priority to: 3

For the input: 9 copper will set batcher pool to: 9
